filerun.js demo

Drop files anywhere on this page.

You can also click here to select files from your computer.

The selection can be limited to a single file.

If you are using Google Chrome, you can also select an entire folder.

This demo uploads the files in very small chunks of 12KB each, to allow you to see how it works without having to upload very large files.
The overall transfer speed drops dramatically with such a small chunk size.
With a default chunk size larger than 2MB, the transfer speed is closer to transferring the file in a single pass.
